# E-Commerce Sales Data Analysis (Python & Pandas)

An end-to-end data analytics project focused on cleaning, aggregating, and visualizing retail performance metrics.

## 🛠️ Tech Stack & Tools
- **Language:** Python 3.x
- **Libraries:** Pandas, Matplotlib, Seaborn, OpenPyXL
- **Environment:** Jupyter Notebook / PyCharm

## 📊 Key Insights Captured
- High-level business KPIs (Total Revenue, Profit margins)
- Product Category breakdowns (Sales vs Profit trends)
- State-wise geographical revenue rankings
